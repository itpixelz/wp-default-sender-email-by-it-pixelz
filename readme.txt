== Description ==

Change your WordPress default sender email address to your own to look like more professional. By default its wordpress@domain.com but right now using this plugin you will be able to change the name of the sender name and the email address from too.

== Installation ==

1. Download the plugin zip file and extract it from zip folder and upload  `wp_default_sender_email_itpixelz` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to settings of the plugin and edit options and save it.
4. Thats it!!! Enjoy :)
